1. GET Gates

Get an array of gates (can be empty, never null) with their numbers and coordinates.

Call example: GET http://localhost:8080/service/gates

Response (JSON):
{
    "gates": [
        {
            "number": 10,
            "lon": 10.4931,
            "lat": 10.4392
        },
        {
            "number": 19,
            "lon": 9.4443,
            "lat": 12.943
        }
    ]
}

2. PUT Gates

Send Captain ID in the system to choose a gate.

If the same Captain ID tries to choose an already taken gate, Success Response is sent.
If a different Captain ID tries to choose an already taken gate, Error Response is sent.

Possible Error Outputs: "Invalid Gate Number", "Invalid Captain ID"

Call example: PUT http://localhost:8080/service/gates/10
Body (JSON):
{
    "captainId": 4
}

Response (JSON):
-- On success --
{
    "success": true
}
-- On failure --
{
    "success": false
    "error": "Invalid Captain ID"
}

3. GET All Messages for First Officer

Get all service messages (like "Gate 4 reserved") for the First Officer ID.
Can return an empty array but never null. All messages are sorted from the newest to the oldest.

Call example: GET http://localhost:8080/service/messages/4

Response:
{
    messages: [
        {
            "time": "11.05.2017 11:44",
            "text": "Gate 4 reserved"
        },
        {
            "time": "09.05.2017 19:11",
            "text": "Gate 19 reserved"
        }
    ]
}
